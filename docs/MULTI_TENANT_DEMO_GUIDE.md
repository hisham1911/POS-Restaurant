## 🔐 System Owner (إدارة جميع المحلات)

```
Email: owner@kasserpro.com
Password: Owner@123
الصلاحيات: إدارة جميع الـ Tenants
```

**يمكنه:**
- عرض جميع المحلات
- إنشاء محلات جديدة
- تفعيل/تعطيل المحلات

## 🎯 سيناريوهات العرض

### سيناريو 1: عرض على صاحب محل أدوات منزلية
```
1. تسجيل الدخول: samy@homeappliances.com / Admin@123
2. عرض المنتجات (أدوات مطبخ، أجهزة كهربائية)
3. إنشاء طلب بيع (خلاط + طقم سكاكين)
4. قبول الدفع (نقدي/بطاقة)
5. طباعة الفاتورة
6. عرض التقارير (المبيعات، المخزون)
```

### سيناريو 2: عرض على صاحب سوبر ماركت
```
1. تسجيل الدخول: karim@supermarket.com / Admin@123
2. عرض المنتجات (بقالة، مشروبات، منظفات)
3. إنشاء طلب بيع متعدد المنتجات
4. تطبيق خصم
5. قبول الدفع
6. عرض تقرير المبيعات اليومية
```

### سيناريو 3: عرض على صاحب مطعم
```
1. تسجيل الدخول: tarek@restaurant.com / Admin@123
2. عرض قائمة الطعام (مشويات، مقبلات)
3. إنشاء طلب (كباب + سلطة + عصير)
4. اختيار نوع الطلب (Dine-In / Takeaway / Delivery)
5. قبول الدفع
6. عرض تقرير الورديات
```

### سيناريو 4: عرض على مستثمر (System Owner)
```
1. تسجيل الدخول: owner@kasserpro.com / Owner@123
2. عرض جميع المحلات (4 محلات)
3. عرض إحصائيات كل محل
4. إنشاء محل جديد
5. تفعيل/تعطيل محل
6. عرض التقارير الشاملة
```


## 💡 نصائح للعرض

### 1. اختر المحل المناسب للعميل
- صاحب محل أدوات → محل أدوات منزلية
- صاحب سوبر ماركت → سوبر ماركت
- صاحب مطعم → مطعم
- مستثمر → System Owner

### 2. ركز على المميزات المهمة للعميل
- **محلات التجزئة:** المخزون، الباركود، الفواتير
- **مطاعم:** أنواع الطلبات، الورديات، التقارير
- **سوبر ماركت:** السرعة، المخزون، العروض
- **مستثمرين:** Multi-tenancy، التقارير الشاملة

### 3. استخدم بيانات واقعية
- جميع المنتجات لها أسعار واقعية
- جميع الحسابات صحيحة (Tax Exclusive 14%)
- جميع البيانات منطقية
