# ✅ Checklist التحقق من التعديلات

> **استخدم هذا الملف للتحقق من أن كل التعديلات تعمل بشكل صحيح**

---

## 🎯 الهدف
التأكد من أن جميع التعديلات على نموذج المنتج وصفحة المنتجات تعمل بشكل صحيح ومتسق.

---

## 📋 Checklist التحقق

### 1. نموذج تعديل المنتج (Product Form Modal)

#### أ. المنتجات التي لها باتشات

- [ ] **افتح منتج له باتشات للتعديل**
  ```
  الخطوات:
  1. اذهب إلى صفحة المنتجات
  2. اختر منتج له باتشات (مثلاً: منتج تم شراؤه من المورد)
  3. اضغط "تعديل"
  ```

- [ ] **تحقق من حقل "سعر البيع"**
  - [ ] يعرض سعر الباتش (مثلاً: 18 ج.م) وليس السعر الأساسي (80 ج.م)
  - [ ] الحقل معطّل (disabled) - لا يمكن التعديل
  - [ ] رسالة تنبيه تظهر: "معطّل لأن المنتج له دفعات مخزون"

- [ ] **تحقق من حقل "سعر التكلفة"**
  - [ ] يعرض تكلفة الباتش
  - [ ] الحقل معطّل (disabled)
  - [ ] رسالة تنبيه تظهر: "معطّل لأن المنتج له دفعات مخزون"

- [ ] **تحقق من صندوق التنبيه الكبير**
  - [ ] يظهر صندوق أصفر (amber) في أعلى قسم التسعير
  - [ ] العنوان: "هذا المنتج له دفعات مخزون (Batches)"
  - [ ] يعرض سعر الباتش الحالي
  - [ ] يعرض السعر الأساسي للمقارنة
  - [ ] يعرض معلومات الباتش القادم (رقم الباتش، تاريخ الانتهاء، السعر)
  - [ ] رابط "شاشة إدارة الدفعات" يعمل

#### ب. المنتجات بدون باتشات

- [ ] **افتح منتج بدون باتشات للتعديل**
  ```
  الخطوات:
  1. اذهب إلى صفحة المنتجات
  2. اختر منتج جديد لم يتم شراؤه من مورد
  3. اضغط "تعديل"
  ```

- [ ] **تحقق من حقل "سعر البيع"**
  - [ ] يعرض السعر الأساسي
  - [ ] الحقل نشط (enabled) - يمكن التعديل
  - [ ] لا توجد رسالة تنبيه

- [ ] **تحقق من حقل "سعر التكلفة"**
  - [ ] يعرض التكلفة الأساسية
  - [ ] الحقل نشط (enabled)
  - [ ] لا توجد رسالة تنبيه

- [ ] **تحقق من عدم وجود صندوق التنبيه**
  - [ ] لا يظهر الصندوق الأصفر

#### ج. حقول الضريبة

- [ ] **تحقق من عدم وجود حقول الضريبة**
  - [ ] لا يوجد حقل "معدل الضريبة (%)"
  - [ ] لا يوجد حقل "السعر شامل الضريبة؟"
  - [ ] قسم التسعير يحتوي فقط على: سعر البيع + سعر التكلفة

#### د. حقل SKU

- [ ] **تحقق من إخفاء حقل SKU**
  - [ ] قسم "الأكواد" يحتوي على حقل واحد فقط: "الباركود"
  - [ ] لا يوجد حقل "SKU"
  - [ ] Grid من عمود واحد (ليس عمودين)

#### هـ. حقل الباركود

- [ ] **تحقق من حقل الباركود**
  - [ ] الحقل موجود وواضح
  - [ ] يمكن إدخال رقم الباركود
  - [ ] Placeholder: "رقم الباركود"

---

### 2. صفحة المنتجات (Products Page)

#### أ. جدول المنتجات

- [ ] **افتح صفحة المنتجات**
  ```
  URL: http://localhost:3000/products
  ```

- [ ] **تحقق من عمود "السعر"**
  - [ ] يعرض سعر واحد فقط
  - [ ] السعر المعروض هو `suggestedPrice` (السعر الفعلي)
  - [ ] لا يوجد سعر ثاني مشطوب
  - [ ] لا يوجد سطرين للسعر

#### ب. المنتجات التي لها باتشات

- [ ] **ابحث عن منتج له باتشات في الجدول**
  - [ ] السعر المعروض = سعر الباتش (مثلاً: 18 ج.م)
  - [ ] ليس السعر الأساسي (80 ج.م)

#### ج. المنتجات بدون باتشات

- [ ] **ابحث عن منتج بدون باتشات في الجدول**
  - [ ] السعر المعروض = السعر الأساسي
  - [ ] العرض نظيف وواضح

---

### 3. نقطة البيع (POS)

#### أ. عرض المنتجات

- [ ] **افتح نقطة البيع**
  ```
  URL: http://localhost:3000/pos
  ```

- [ ] **تحقق من أسعار المنتجات**
  - [ ] كل منتج يعرض `suggestedPrice`
  - [ ] المنتجات التي لها باتشات تعرض سعر الباتش
  - [ ] المنتجات بدون باتشات تعرض السعر الأساسي

#### ب. إضافة منتج للسلة

- [ ] **أضف منتج له باتش للسلة**
  ```
  الخطوات:
  1. اختر منتج له باتش
  2. اضغط عليه لإضافته للسلة
  ```

- [ ] **تحقق من السعر في السلة**
  - [ ] السعر في السلة = سعر الباتش
  - [ ] الإجمالي يحسب بناءً على سعر الباتش
  - [ ] الضريبة تُحسب بشكل صحيح

#### ج. إتمام الطلب

- [ ] **أتمم طلب يحتوي على منتج له باتش**
  ```
  الخطوات:
  1. أضف منتج للسلة
  2. اضغط "إتمام الطلب"
  3. اختر طريقة الدفع
  4. أتمم الطلب
  ```

- [ ] **تحقق من الفاتورة**
  - [ ] السعر في الفاتورة = سعر الباتش
  - [ ] الإجمالي صحيح
  - [ ] الضريبة صحيحة

---

### 4. حسابات السلة (Cart Pricing)

#### أ. حساب السعر الصافي

- [ ] **أضف منتج للسلة وتحقق من الحسابات**
  ```
  مثال:
  - منتج: تيشيرت
  - سعر الباتش: 18 ج.م
  - الكمية: 2
  ```

- [ ] **تحقق من:**
  - [ ] السعر الصافي = 18 × 2 = 36 ج.م
  - [ ] الضريبة = 36 × (14 / 100) = 5.04 ج.م
  - [ ] الإجمالي = 36 + 5.04 = 41.04 ج.م

---

### 5. إدارة الدفعات (Batch Management)

#### أ. تعديل سعر الباتش

- [ ] **اذهب إلى شاشة إدارة الدفعات**
  ```
  URL: http://localhost:3000/product-batches
  ```

- [ ] **عدّل سعر باتش**
  ```
  الخطوات:
  1. اختر منتج
  2. عدّل سعر البيع للباتش
  3. احفظ التعديل
  ```

- [ ] **تحقق من التحديث التلقائي**
  - [ ] ارجع لصفحة المنتجات
  - [ ] السعر في الجدول تحدث
  - [ ] افتح نموذج تعديل المنتج
  - [ ] السعر في النموذج تحدث
  - [ ] افتح نقطة البيع
  - [ ] السعر في POS تحدث

## 🐛 اختبارات الحالات الخاصة

### حالة 1: منتج بدون باتشات → إضافة باتش

- [ ] **الخطوات:**
  ```
  1. أنشئ منتج جديد بسعر 100 ج.م
  2. تحقق من أن السعر في كل الشاشات = 100 ج.م
  3. أضف باتش للمنتج بسعر 80 ج.م
  4. تحقق من أن السعر في كل الشاشات = 80 ج.م
  ```

### حالة 2: منتج له باتش → حذف الباتش

- [ ] **الخطوات:**
  ```
  1. اختر منتج له باتش (سعر الباتش 18 ج.م، السعر الأساسي 80 ج.م)
  2. احذف الباتش
  3. تحقق من أن السعر في كل الشاشات = 80 ج.م (السعر الأساسي)
  ```

### حالة 3: منتج له أكثر من باتش

- [ ] **الخطوات:**
  ```
  1. أضف باتشين لنفس المنتج:
     - باتش 1: ينتهي 2026-06-01، سعر 15 ج.م
     - باتش 2: ينتهي 2026-05-15، سعر 18 ج.م
  2. تحقق من أن النظام يختار الباتش الأقرب للانتهاء (FEFO)
  3. السعر المعروض = 18 ج.م (باتش 2)
  ```

## 🔧 اختبارات التكامل

### اختبار 1: دورة حياة كاملة

- [ ] **الخطوات:**
  ```
  1. أنشئ منتج جديد: "تيشيرت أحمر - M"
     - السعر: 100 ج.م
     - التكلفة: 50 ج.م
     - الباركود: 6281000123451
  
  2. تحقق من صفحة المنتجات:
     - السعر المعروض = 100 ج.م
  
  3. أضف باتش من المورد:
     - الكمية: 50
     - سعر البيع: 80 ج.م
     - التكلفة: 40 ج.م
  
  4. تحقق من صفحة المنتجات:
     - السعر المعروض = 80 ج.م
  
  5. افتح نموذج تعديل المنتج:
     - سعر البيع = 80 ج.م (معطّل)
     - التكلفة = 40 ج.م (معطّل)
     - صندوق التنبيه يظهر
  
  6. اذهب لنقطة البيع:
     - السعر المعروض = 80 ج.م
  
  7. بع 10 قطع:
     - السعر في الفاتورة = 80 ج.م
     - الإجمالي = 800 ج.م
     - الضريبة = 112 ج.م
     - المجموع الكلي = 912 ج.م
  
  8. تحقق من المخزون:
     - الكمية المتبقية = 40
  ```

### اختبار 2: مسح الباركود

- [ ] **الخطوات:**
  ```
  1. أضف منتج بباركود: 6281000123451
  2. في نقطة البيع، استخدم ماسح الباركود (أو اكتب الباركود)
  3. تحقق من أن المنتج يضاف للسلة تلقائياً
  4. تحقق من أن السعر صحيح
  ```
