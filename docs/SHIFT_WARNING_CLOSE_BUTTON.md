### 2. **إضافة State Management في `POSPage.tsx`**

const [dismissedWarning, setDismissedWarning] = useState(false);

// ✅ إعادة تعيين الإغلاق عند تغيير الرسالة

  setDismissedWarning(false);

    />
  </div>
)}

## 🎯 كيف يعمل النظام

### 1. **عند ظهور الإشعار**

```
User opens POS → Shift warning appears
                      ↓
         [Warning Banner with X button]
```

### 2. **عند الضغط على X**

```
User clicks X → setDismissedWarning(true)
                      ↓
         Warning banner disappears
```

### 3. **عند تغيير الرسالة**

```
Warning message changes → useEffect triggers
                              ↓
                  setDismissedWarning(false)
                              ↓
                  Warning appears again
```

**السبب:** إذا تغيرت رسالة التحذير (مثلاً من "تحذير" إلى "تحذير شديد")، يجب أن يظهر الإشعار مرة أخرى.

## 📝 ملاحظات مهمة

### 1. **الإغلاق مؤقت (Session-based)**

الإغلاق يستمر فقط خلال الجلسة الحالية:
- ✅ لو المستخدم أغلق الإشعار → يختفي
- ✅ لو المستخدم عمل refresh للصفحة → يظهر مرة أخرى
- ✅ لو تغيرت رسالة التحذير → يظهر مرة أخرى

**السبب:** التحذيرات مهمة ويجب أن يراها المستخدم، لكن نعطيه خيار إخفائها مؤقتًا.

### 2. **Polling Interval**

| الصفحة | Polling Interval |
|--------|------------------|
| POSPage | 10 دقائق |
| POSWorkspacePage | 10 دقائق |
| ShiftPage | 5 دقائق |

### 3. **مستويات التحذير**

| المستوى | الشرط | اللون |
|---------|-------|-------|
| Warning | 8-12 ساعة | أصفر 🟡 |
| Critical | > 12 ساعة | أحمر 🔴 |
