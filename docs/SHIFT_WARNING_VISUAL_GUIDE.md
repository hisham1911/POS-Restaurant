## ✅ Accessibility

```
<button
  onClick={onClose}
  aria-label="إغلاق التحذير"     ← Screen reader
  title="إغلاق التحذير"          ← Tooltip
  className="..."
>
  <XCircle className="w-6 h-6" />
</button>
```

**Features:**
- ✅ `aria-label` للقارئات الشاشة
- ✅ `title` tooltip عند hover
- ✅ حجم كبير للضغط السهل (40×40px)
- ✅ ألوان واضحة ومتباينة

## 🔍 التفاصيل التقنية

### CSS Classes

```tsx
// Before
className="p-1 rounded-lg hover:bg-opacity-20 transition-colors"

// After
className={clsx(
  "p-2 rounded-lg hover:bg-opacity-20 transition-colors flex-shrink-0",
  isCritical
    ? "hover:bg-red-600 text-red-600"
    : "hover:bg-yellow-600 text-yellow-600"
)}
```

### Icon Size

```tsx
// Before
<XCircle className="w-5 h-5" />  // 20px

// After
<XCircle className="w-6 h-6" />  // 24px
```
