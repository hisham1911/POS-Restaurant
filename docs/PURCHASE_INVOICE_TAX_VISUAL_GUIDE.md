## 🔄 User Flow - سيناريوهات الاستخدام

### السيناريو 1: مورد خاضع للضريبة

```
┌─────────────────────────────────────────────────────────────┐
│ 1. المستخدم يفتح صفحة إنشاء فاتورة جديدة                   │
│    → Checkbox "تطبيق الضريبة" مُفعّل افتراضيًا ✅           │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│ 2. المستخدم يختار المورد: "شركة ABC"                       │
│    → المورد خاضع للضريبة                                   │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│ 3. المستخدم يضيف منتجات:                                   │
│    - منتج أ: 10 × 100 = 1000 ج.م                           │
│    - منتج ب: 5 × 200 = 1000 ج.م                            │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│ 4. النظام يحسب تلقائيًا:                                    │
│    - Subtotal: 2000 ج.م                                    │
│    - Tax (14%): 280 ج.م                                    │
│    - Total: 2280 ج.م                                       │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│ 5. المستخدم يضغط "حفظ"                                     │
│    → الفاتورة تُحفظ بـ isTaxEnabled = true                 │
│    → Total = 2280 ج.م ✅                                    │
└─────────────────────────────────────────────────────────────┘
```
