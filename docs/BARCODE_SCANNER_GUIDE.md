## 🎯 كيف يعمل النظام حالياً

### في نقطة البيع (POS)

```
┌─────────────────────────────────────────┐
│  🔍 ابحث بالاسم أو الباركود أو SKU    │ ← حقل البحث
│  [                                   ]  │
└─────────────────────────────────────────┘
         ↓
    جهاز الباركود يكتب الرقم هنا تلقائياً
         ↓
    تضغط Enter (أو الجهاز يضغطه تلقائياً)
         ↓
    النظام يبحث عن المنتج بالباركود
         ↓
    المنتج يضاف للسلة مباشرة ✅
```

## 🔧 كيفية الاستخدام

### الخطوة 1: توصيل الجهاز

```
1. وصّل جهاز الباركود في USB
2. Windows هيتعرف عليه تلقائياً (Plug & Play)
3. مفيش برامج أو drivers محتاجها
4. الجهاز بيشتغل زي الكيبورد العادي
```

### الخطوة 2: في نقطة البيع

```
1. افتح نقطة البيع
2. الكيرسر هيكون في حقل البحث تلقائياً
3. امسح الباركود بالجهاز
4. المنتج يضاف للسلة مباشرة ✅
```

### الخطوة 3: إتمام البيع

```
1. امسح كل المنتجات
2. اضغط F2 (أو زر "إتمام الطلب")
3. اختر طريقة الدفع
4. اطبع الفاتورة
```

## ⚙️ الإعدادات التلقائية

### ما يحدث تلقائياً:

```
✅ حقل البحث يكون نشط (focused) دائماً
✅ جهاز الباركود يكتب الرقم في الحقل
✅ جهاز الباركود يضغط Enter تلقائياً
✅ النظام يبحث عن المنتج بالباركود
✅ المنتج يضاف للسلة مباشرة
✅ الكيرسر يرجع لحقل البحث تلقائياً
✅ جاهز لمسح المنتج التالي
```

### الكود المسؤول عن ذلك:

```typescript
// في POSWorkspacePage.tsx - السطر 377-392

const handleSearchSubmit = useCallback((value: string) => {
  const trimmedValue = value.trim();
  if (!trimmedValue) return;

  // البحث بالباركود أو SKU أو الاسم
  const foundProduct = products.find(
    (product) =>
      (product.barcode && 
       product.barcode.toLowerCase() === trimmedValue.toLowerCase()) ||
      (product.sku && 
       product.sku.toLowerCase() === trimmedValue.toLowerCase()) ||
      product.name.toLowerCase() === trimmedValue.toLowerCase()
  );

  if (foundProduct) {
    const added = handleAddProductToCart(foundProduct, { showToast: true });
    if (added) {
      setSearchInput("");           // ✅ مسح حقل البحث
      searchInputRef.current?.focus(); // ✅ الرجوع للحقل
      setShowCatalog(false);
    }
  } else {
    toast.error(`لم يتم العثور على منتج: ${trimmedValue}`);
  }
}, [handleAddProductToCart, products]);

// عند الضغط على Enter
const handleSearchKeyDown = (event: React.KeyboardEvent) => {
  if (event.key === "Enter") {
    event.preventDefault();
    handleSearchSubmit(searchInput); // ✅ البحث والإضافة
  }
};
```

## 🔍 طرق البحث المدعومة

النظام يبحث في 3 حقول:

```
1. الباركود (Barcode)
   ├─ مثال: 6281000123456
   └─ الأكثر استخداماً ✅

2. SKU (Stock Keeping Unit)
   ├─ مثال: TSH-RED-M
   └─ للاستخدام الداخلي

3. اسم المنتج (Name)
   ├─ مثال: Panadol
   └─ للبحث اليدوي
```

## 🔫 أنواع أجهزة الباركود

### 1. الماسح السلكي (USB Wired Scanner)

```
المميزات:
✅ رخيص (300-800 ج.م)
✅ لا يحتاج شحن
✅ موثوق جداً
✅ Plug & Play

العيوب:
❌ السلك ممكن يعيق الحركة

الاستخدام:
├─ وصّل في USB
└─ امسح الباركود مباشرة
```

### 2. الماسح اللاسلكي (Wireless Scanner)

```
المميزات:
✅ حرية الحركة
✅ مريح للاستخدام
✅ مدى يصل لـ 10-30 متر

العيوب:
❌ أغلى (800-2000 ج.م)
❌ يحتاج شحن

الاستخدام:
├─ وصّل الـ USB Receiver في الكمبيوتر
├─ شغّل الماسح
└─ امسح الباركود من أي مكان
```

### 3. ماسح 2D (للباركود ثنائي الأبعاد)

```
المميزات:
✅ يقرأ الباركود العادي (1D)
✅ يقرأ الباركود ثنائي الأبعاد (2D)
✅ يقرأ QR Code

العيوب:
❌ أغلى (1500-4000 ج.م)

الاستخدام:
├─ مفيد للمستقبل
└─ لو الأدوية بدأت تستخدم 2D Barcode
```

## 💡 نصائح مهمة

### 1. موضع الماسح

```
✅ الصح:
├─ قريب من الكاشير
├─ في متناول اليد
└─ زاوية مريحة

❌ الغلط:
├─ بعيد عن الكاشير
├─ محتاج حركة كثيرة
└─ زاوية غير مريحة
```

### 2. الإضاءة

```
✅ الصح:
├─ إضاءة جيدة
└─ لا انعكاسات على الباركود

❌ الغلط:
├─ إضاءة ضعيفة
└─ انعكاسات قوية
```

### 3. حالة الباركود

```
✅ الصح:
├─ باركود واضح
├─ غير ممزق
└─ غير مطموس

❌ الغلط:
├─ باركود ممزق
├─ مطموس
└─ غير واضح
```

### 4. المسافة

```
معظم الماسحات:
├─ المسافة المثالية: 5-15 سم
├─ أقرب من 5 سم: قد لا يقرأ
└─ أبعد من 15 سم: قد لا يقرأ
```

## 🐛 حل المشاكل الشائعة

### المشكلة 1: الماسح لا يعمل

```
الحلول:
1. تأكد من توصيل USB صحيح
2. جرب منفذ USB آخر
3. أعد تشغيل الكمبيوتر
4. تأكد من أن الماسح مشحون (لو لاسلكي)
5. تأكد من تشغيل الماسح (زر الباور)
```

### المشكلة 2: الماسح يكتب أرقام غريبة

```
السبب:
├─ الماسح مضبوط على لغة/ترميز مختلف

الحل:
├─ راجع دليل الماسح
├─ اضبطه على "USB Keyboard Mode"
└─ أو "HID Mode"
```

### المشكلة 3: الماسح يقرأ لكن المنتج لا يضاف

```
الأسباب المحتملة:
1. الباركود غير موجود في النظام
   └─ الحل: أضف المنتج بالباركود الصحيح

2. الباركود مكتوب غلط في النظام
   └─ الحل: عدّل الباركود في بيانات المنتج

3. الماسح يضيف مسافات أو أحرف زيادة
   └─ الحل: اضبط إعدادات الماسح
```

### المشكلة 4: الماسح بطيء

```
الأسباب:
1. الماسح قديم أو رخيص جداً
2. الباركود غير واضح
3. الإضاءة سيئة

الحل:
├─ نظف عدسة الماسح
├─ حسّن الإضاءة
└─ فكر في ترقية الماسح
```

## 🎓 التدريب السريع للكاشير

```
1. افتح نقطة البيع
```

```
├─ إتمام الطلبات
```
