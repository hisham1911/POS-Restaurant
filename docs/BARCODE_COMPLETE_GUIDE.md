## ما هو الباركود؟

### التعريف البسيط

```
الباركود = رقم فريد لكل منتج
├─ يُطبع على المنتج من المصنع
├─ يُقرأ بجهاز مسح ضوئي
└─ يُستخدم للبيع السريع
```

### القاعدة الذهبية

```
نفس المنتج = نفس الباركود
(حتى لو 100 علبة!)

مقاس/حجم/تركيز مختلف = باركود مختلف
```

### مثال عملي

```
10 علب Panadol 500mg (20 قرص)
├─ علبة 1: باركود 6281000123456
├─ علبة 2: باركود 6281000123456
├─ علبة 3: باركود 6281000123456
├─ ... (كلهم نفس الباركود)
└─ علبة 10: باركود 6281000123456

✅ كلهم نفس الباركود!

لكن:
Panadol 500mg (10 أقراص): 6281000123457 ← باركود مختلف!
```

## الباركود في إضافة المنتجات

### ✅ نعم! يمكنك استخدام جهاز الباركود

```
┌─────────────────────────────────────────┐
│  نموذج إضافة منتج جديد                 │
├─────────────────────────────────────────┤
│  الاسم: [Panadol 500mg]                │
│                                         │
│  الباركود: [                        ]  │ ← اضغط هنا
│  💡 يمكنك استخدام جهاز الباركود       │
│                                         │
│  السعر: [50]                           │
└─────────────────────────────────────────┘
         ↓
    امسح الباركود بالجهاز
         ↓
    الرقم يكتب تلقائياً ✅
```

### الخطوات

```
1. افتح نموذج المنتج
2. املأ الاسم والتصنيف
3. اضغط في حقل الباركود
4. امسح الباركود من العلبة
5. الرقم يظهر تلقائياً
6. أكمل باقي البيانات
7. احفظ ✅

الوقت: 30-60 ثانية لكل منتج
```

## الباركود في فاتورة الشراء

### ✅ نعم! يعمل تماماً في فواتير الشراء

```
┌─────────────────────────────────────────┐
│  فاتورة شراء جديدة                     │
├─────────────────────────────────────────┤
│  المورد: [اختر المورد]                 │
│                                         │
│  المنتج: [                          ]  │ ← اضغط هنا
│  💡 يمكنك مسح الباركود مباشرة         │
│                                         │
│  الكمية: [10]                          │
│  سعر الشراء: [30]                      │
└─────────────────────────────────────────┘
         ↓
    امسح الباركود بالجهاز
         ↓
    المنتج يُختار تلقائياً ✅
         ↓
    سعر البيع يملأ تلقائياً
         ↓
    أدخل الكمية وسعر الشراء
```

### سيناريو عملي

```
إضافة فاتورة شراء:

1. افتح فاتورة شراء جديدة
2. اختر المورد والتاريخ
3. لكل منتج:
   ├─ امسح الباركود (ثانية واحدة)
   ├─ المنتج يُختار تلقائياً
   ├─ أدخل الكمية: 100
   ├─ أدخل سعر الشراء: 30 ج.م
   ├─ (اختياري) رقم الباتش: LOT2024-A123
   ├─ (اختياري) تاريخ الانتهاء: 2026-12-31
   └─ اضغط "إضافة" ✅
4. راجع الفاتورة
5. احفظ ✅

الوقت: 2-3 دقائق لفاتورة كاملة!
```

## الباركود في نقطة البيع

### ✅ نعم! يعمل تلقائياً بدون أي إعدادات

```
┌─────────────────────────────────────────┐
│  🔍 ابحث بالاسم أو الباركود           │
│  [                                   ]  │ ← الكيرسر هنا تلقائياً
└─────────────────────────────────────────┘
         ↓
    امسح الباركود بالجهاز
         ↓
    المنتج يضاف للسلة تلقائياً ✅
         ↓
    الكيرسر يرجع للحقل تلقائياً
         ↓
    جاهز لمسح المنتج التالي
```

### سيناريو عملي

```
الكاشير في الصيدلية:

1. يفتح نقطة البيع
2. يمسح باركود Panadol: 6281000123456
   └─ Panadol يضاف للسلة (50 ج.م)
3. يمسح باركود Aspirin: 6281000123457
   └─ Aspirin يضاف للسلة (30 ج.م)
4. يمسح باركود Panadol مرة تانية
   └─ الكمية تزيد (Panadol × 2)
5. يضغط F2 (إتمام الطلب)
6. يختار "نقدي"
7. يطبع الفاتورة

الوقت الكلي: أقل من دقيقة! ⚡
```

## جهاز الباركود

### الأنواع والأسعار

| النوع | السعر | المميزات | التوصية |
|-------|-------|----------|----------|
| **سلكي USB** | 300-800 ج.م | رخيص، موثوق، Plug & Play | **الأفضل للبداية** ⭐ |
| **لاسلكي** | 800-2000 ج.م | حرية الحركة، مريح | للمحلات الكبيرة |
| **2D Scanner** | 1500-4000 ج.م | يقرأ كل الأنواع + QR | للمستقبل |

### كيفية الاستخدام

```
1. وصّل الجهاز في USB
   └─ Windows هيتعرف عليه تلقائياً

2. افتح نقطة البيع (أو نموذج المنتج)
   └─ الكيرسر هيكون في حقل البحث/الباركود

3. امسح الباركود
   └─ الجهاز يكتب الرقم ويضغط Enter تلقائياً

4. المنتج يضاف/يحفظ تلقائياً ✅
   └─ جاهز للمنتج التالي

مفيش:
❌ إعدادات معقدة
❌ برامج إضافية
❌ تعديلات في الكود
```

## الأسئلة الشائعة

### س1: هل كل علبة لها باركود مختلف؟

```
❌ لا!

نفس المنتج = نفس الباركود:
├─ 10 علب Panadol 500mg
└─ كلهم باركود: 6281000123456

مقاس/حجم مختلف = باركود مختلف:
├─ Panadol 500mg (20 قرص): 6281000123456
└─ Panadol 500mg (10 أقراص): 6281000123457
```

### س2: هل أحتاج إعدادات خاصة؟

```
❌ لا!

فقط:
1. وصّل الجهاز في USB
2. افتح النظام
3. امسح الباركود
4. يشتغل تلقائياً ✅
```

### س3: هل يعمل في إضافة المنتجات؟

```
✅ نعم!

في نموذج المنتج:
1. اضغط في حقل الباركود
2. امسح الباركود
3. الرقم يكتب تلقائياً
4. أكمل باقي البيانات
5. احفظ ✅
```

### س4: هل يعمل في نقطة البيع؟

```
✅ نعم!

في POS:
1. افتح نقطة البيع
2. امسح الباركود
3. المنتج يضاف للسلة تلقائياً
4. امسح المنتج التالي
5. سريع جداً! ⚡
```

### س5: ماذا لو المنتج بدون باركود؟

```
الحلول:
1. اطبع باركود خاص بيك
   └─ طابعة باركود (1000-3000 ج.م)

2. استخدم البحث اليدوي
   └─ اكتب اسم المنتج في POS

3. اترك الحقل فارغ
   └─ الباركود اختياري (مش إلزامي)
```

### س6: لو عندي دفعتين من نفس الدواء؟

```
في KasserPro:
├─ المنتج واحد (نفس الباركود)
└─ باتشين مختلفين

مثال:
منتج: Panadol 500mg
├─ الباركود: 6281000123456 (ثابت)
├─ باتش 1: LOT-A (5 علب، ينتهي 2026-12)
├─ باتش 2: LOT-B (5 علب، ينتهي 2027-03)
└─ الإجمالي: 10 علب

في نقطة البيع:
├─ مسح الباركود: 6281000123456
└─ النظام يختار الباتش الأقرب للانتهاء (FEFO)
```

## 🎯 الخلاصة النهائية

### النظام جاهز 100%!

```
✅ الباركود يعمل في إضافة المنتجات
✅ الباركود يعمل في فاتورة الشراء
✅ الباركود يعمل في نقطة البيع
✅ جهاز الباركود Plug & Play
✅ لا إعدادات معقدة
✅ لا برامج إضافية
✅ سريع وسهل ومريح
```

### ما تحتاجه للبدء

```
1. جهاز باركود USB (300-800 ج.م)
2. وصّله في الكمبيوتر
3. افتح النظام
4. ابدأ الاستخدام! 🚀

الوقت المطلوب: 5 دقائق فقط!
```

### الفوائد

```
السرعة:
├─ إضافة منتج: 30-60 ثانية
├─ بيع منتج: 1-2 ثانية
└─ 10× أسرع من الكتابة اليدوية

الدقة:
├─ لا أخطاء إملائية
├─ لا أخطاء في الأرقام
└─ 100% دقيق

الراحة:
├─ لا كتابة يدوية
├─ لا بحث يدوي
└─ مريح جداً للكاشير
```
