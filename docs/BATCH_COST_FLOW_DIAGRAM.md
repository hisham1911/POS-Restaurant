# 🔄 مخطط تدفق تكلفة الباتشات في KasserPro

## 📊 دورة حياة سعر التكلفة

```
┌─────────────────────────────────────────────────────────────────┐
│                    1️⃣ فاتورة الشراء                            │
│                   (Purchase Invoice)                            │
└─────────────────────────────────────────────────────────────────┘
                              │
                              │ CreatePurchaseInvoiceItemRequest
                              │ { PurchasePrice: 50 ج.م }
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│              PurchaseInvoiceService.ConfirmAsync()              │
│                                                                 │
│  ✅ Validation: PurchasePrice >= 0                             │
│  ✅ Create ProductBatch:                                       │
│     - CostPrice = item.PurchasePrice (50 ج.م)                 │
│     - SellingPrice = item.SellingPrice (70 ج.م)               │
│     - Quantity = 10                                            │
│                                                                 │
│  ✅ Update Product.AverageCost (Weighted Average):             │
│     AverageCost = (oldValue + newValue) / (oldQty + newQty)    │
└─────────────────────────────────────────────────────────────────┘
                              │
                              │ ProductBatch Created
                              │ { CostPrice: 50, SellingPrice: 70 }
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                    2️⃣ عملية البيع                              │
│                      (Order)                                    │
└─────────────────────────────────────────────────────────────────┘
                              │
                              │ AddOrderItemRequest
                              │ { ProductId, Quantity, BatchId? }
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│              OrderService.CreateAsync() / AddItemAsync()        │
│                                                                 │
│  ✅ ResolveBatchSaleSnapshotAsync():                           │
│     Priority:                                                   │
│     1. ProductBatch.CostPrice (if BatchId selected)            │
│     2. Product.AverageCost                                     │
│     3. Product.Cost                                            │
│     4. 0 (fallback)                                            │
│                                                                 │
│  ✅ Create OrderItem:                                          │
│     - UnitCost = batchCost (50 ج.م) ← SNAPSHOT                │
│     - UnitPrice = 70 ج.م                                      │
│     - BatchId = 123                                            │
│     - Quantity = 2                                             │
└─────────────────────────────────────────────────────────────────┘
                              │
                              │ OrderItem Created
                              │ { UnitCost: 50, UnitPrice: 70 }
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                    3️⃣ إتمام البيع                              │
│                  (Complete Order)                               │
└─────────────────────────────────────────────────────────────────┘
                              │
                              │ CompleteOrderRequest
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│              OrderService.CompleteAsync()                       │
│                                                                 │
│  ✅ Decrement Stock from Batch                                 │
│  ✅ OrderItem.UnitCost remains UNCHANGED (50 ج.م)             │
│  ✅ Create StockMovement record                                │
└─────────────────────────────────────────────────────────────────┘
                              │
                              │ Order Completed
                              │ { Status: Completed }
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                    4️⃣ التقارير المالية                        │
│                   (Financial Reports)                           │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
        ┌─────────────────────┴─────────────────────┐
        │                                           │
        ▼                                           ▼
┌──────────────────┐                    ┌──────────────────┐
│ Profit & Loss    │                    │ Product Movement │
│                  │                    │                  │
│ Cost = Σ(        │                    │ Cost = Σ(        │
│   OrderItem      │                    │   OrderItem      │
│   .UnitCost      │                    │   .UnitCost      │
│   * Quantity     │                    │   * Quantity     │
│ )                │                    │ )                │
│                  │                    │                  │
│ = 50 × 2         │                    │ = 50 × 2         │
│ = 100 ج.م        │                    │ = 100 ج.م        │
└──────────────────┘                    └──────────────────┘
        │                                           │
        └─────────────────────┬─────────────────────┘
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                    ✅ النتيجة النهائية                         │
│                                                                 │
│  Revenue: 70 × 2 = 140 ج.م                                     │
│  Cost:    50 × 2 = 100 ج.م                                     │
│  Profit:  140 - 100 = 40 ج.م                                   │
│  Margin:  (40 / 140) × 100 = 28.57%                            │
└─────────────────────────────────────────────────────────────────┘
```

                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐

                              │

## 📊 مقارنة: مع وبدون Snapshot

### ❌ بدون Snapshot (خطأ)

```
البيع (اليوم 1):
  Cost = Product.Cost (50 ج.م)

شراء جديد (اليوم 2):
  Product.Cost = 60 ج.م

التقرير (اليوم 3):
  Cost = Product.Cost (60 ج.م) ← خطأ! السعر تغير
  Profit = Revenue - 60 ← غير دقيق
```

### ✅ مع Snapshot (صحيح)

```
البيع (اليوم 1):
  OrderItem.UnitCost = 50 ج.م (SNAPSHOT)

شراء جديد (اليوم 2):
  Product.Cost = 60 ج.م

التقرير (اليوم 3):
  Cost = OrderItem.UnitCost (50 ج.م) ← صحيح! القيمة محفوظة
  Profit = Revenue - 50 ← دقيق 100%
```

## 🎯 الخلاصة المرئية

```
┌─────────────────────────────────────────────────────────────────┐
│                    🔒 Snapshot Pattern                          │
│                                                                 │
│  Purchase → ProductBatch.CostPrice (50 ج.م)                    │
│                    ↓                                            │
│  Sale → OrderItem.UnitCost (50 ج.م) ← FROZEN                   │
│                    ↓                                            │
│  Report → Uses OrderItem.UnitCost (50 ج.م) ← ACCURATE          │
│                                                                 │
│  ✅ التقارير دقيقة تاريخياً                                   │
│  ✅ لا تتأثر بتغيير الأسعار                                   │
│  ✅ كل بيع يحتفظ بتكلفته الصحيحة                              │
└─────────────────────────────────────────────────────────────────┘
```

## 🔍 نقاط التحقق (Checkpoints)

| المرحلة | الحقل | القيمة | الحالة |
|---------|-------|--------|--------|
| 1. Purchase | `PurchaseInvoiceItem.PurchasePrice` | 50 ج.م | ✅ Input |
| 2. Batch | `ProductBatch.CostPrice` | 50 ج.م | ✅ Stored |
| 3. Sale | `OrderItem.UnitCost` | 50 ج.م | ✅ Snapshot |
| 4. Report | `Sum(OrderItem.UnitCost * Quantity)` | 100 ج.م | ✅ Accurate |

---

**النتيجة:** ✅ النظام يحافظ على سلامة البيانات في كل مرحلة
